# /research-competitor - 헬스케어 IT 경쟁사 분석

## 개요

미국 RPM/CCM/APCM/HRA/AWV/VBC 시장의 주요 경쟁사를 체계적으로 분석한다. 웹사이트, 서비스 범위, 메시징 전략, UX 패턴을 조사하여 HicareNet의 경쟁 포지셔닝 전략을 도출한다.

## 분석 대상

`references/competitor-list.md`에 정리된 10-15개 경쟁사를 분석한다. 목록에 없는 경쟁사가 조사 중 발견되면 추가한다.

## 분석 워크플로우

### 경쟁사별 분석 (각 경쟁사에 대해 반복)

**1단계: 웹사이트 수집**
- WebFetch로 홈페이지, Solutions 페이지, About 페이지, Pricing 페이지(있는 경우)를 가져온다
- 차단 시 WebSearch로 대안 검색. `[PARTIAL]` 표기

**2단계: 서비스 커버리지 매핑**
각 경쟁사가 다음 서비스를 제공하는지 확인:
- RPM / CCM / APCM / HRA / AWV / VBC / FHIR / 기타

**3단계: 메시징 전략 분석**
- 아웃컴 중심 vs 기능 중심 메시징 판별
- 타겟 청중 (병원, 클리닉, ACO, SNF 등)
- 신뢰 구축 전략 (인증, 수상, 사례 연구, 파트너십)
- 헤드라인/히어로 카피 패턴
- CTA 유형과 배치

**4단계: UX/구조 패턴**
- 사이트맵 (주요 페이지 구조)
- 네비게이션 패턴 (메가메뉴, 드롭다운 등)
- 솔루션 페이지 레이아웃 (탭, 아코디언, 카드 등)
- 모바일 대응 여부
- 리소스 섹션 구성 (블로그, 백서, 웨비나 등)

**5단계: 차별화 주장 기록**
- 경쟁사가 스스로 주장하는 고유 강점
- 정량적 주장 (환자 수, 디바이스 수, 성과 지표 등)
- 기술 차별화 (AI, 인터페이스, 통합 등)

### 종합 분석

**비교 매트릭스 작성**
행: 경쟁사, 열: RPM/CCM/APCM/HRA/AWV/VBC/FHIR/다국어/가격투명성/디바이스수/환자수
각 셀에 O(지원)/X(미지원)/?(불확실) 기입

**메시징 패턴 종합**
- 업계 공통 패턴 vs HicareNet 차별화 기회
- 가장 효과적인 메시징 전략 식별
- content-strategist가 참고할 톤/CTA 권고

**UX 패턴 종합**
- 업계 표준 페이지 구조
- visual-designer가 참고할 레이아웃/인터랙션 권고

**HicareNet 포지셔닝**
- 경쟁 우위 (다국어 7개 국어, 120+ 디바이스, 게이트웨이 하드웨어)
- 경쟁 열위 (상대적으로 작은 규모, 브랜드 인지도)
- 차별화 전략 권고

## 품질 기준

- 분석은 사실 기반이어야 한다. 경쟁사를 폄하하거나 과대평가하지 않는다
- 각 분석 항목에 출처 URL을 포함한다
- 접근 불가 시 `[PARTIAL: 직접 접근 차단, 검색 결과 기반]` 표기
- 미공개 정보(가격, 고객 수)는 `[NOT PUBLIC]` 표기
- 인수합병된 기업은 현재 브랜드와 이전 브랜드 모두 기록
