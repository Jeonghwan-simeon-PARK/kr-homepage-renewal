# /content - Healthcare Content Generator

Generate marketing copy for HicareNet website pages following healthcare B2B best practices.

## Usage
```
/content [page] [section]
/content home hero
/content solutions rpm
/content about company-story
/content all                    # Generate all page content
```

## Content Principles

1. **Outcome over Feature**: "Reduce readmissions" not "Real-time dashboard"
2. **Audience-First**: Lead with provider pain points
3. **Compliance as Trust**: Weave naturally, don't isolate
4. **Concrete Numbers**: 200+ clinics, 15K+ patients, 20+ years, 7 languages
5. **Clinical Credibility**: Reference CPT codes, CMS guidelines

## Service Content Frameworks

### RPM (Remote Patient Monitoring)
- **Core message**: Enable proactive care between visits
- **Value props**: Reduce readmissions, improve outcomes, increase revenue
- **CPT codes**: 99453, 99454, 99457, 99458
- **Differentiator**: Integrated device + software platform

### CCM (Chronic Care Management)
- **Core message**: Coordinate care for chronic conditions
- **Value props**: Multilingual engagement (7 languages), personalized care plans
- **CPT codes**: 99490, 99491, 99487, 99489
- **Differentiator**: Multilingual capability, culturally aware care

### AWV (Annual Wellness Visits)
- **Core message**: Streamline preventive care delivery
- **Value props**: Automated HRA, Medicare compliance, prevention plans
- **CPT codes**: G0438, G0439
- **Differentiator**: Digital HRA automation

### VBC (Value-Based Care)
- **Core message**: Succeed in value-based payment models
- **Value props**: Quality dashboards, HEDIS/Stars optimization, care gap closure
- **Differentiator**: Comprehensive quality measure tracking

### FHIR Integration
- **Core message**: Seamless EMR interoperability
- **Value props**: Standard HL7 FHIR R4, reduce manual data entry, real-time sync
- **Timeline**: H1 2026
