# 문의

## 페이지 목적
투자자, 채용 지원자, 파트너가 용도에 맞는 문의를 보낼 수 있도록 유형별 폼을 제공하고, 담당자에게 자동 라우팅한다.

## 콘텐츠

### 섹션 (A): 문의 안내

**페이지 타이틀**
문의하기

**인트로 카피**
하이케어넷에 대해 궁금한 점이 있으시면 아래 폼을 통해 문의해 주세요. 문의 유형에 따라 담당자가 확인 후 회신드리겠습니다.


### 섹션 (B): 문의 폼

**문의 유형 선택 (라디오 버튼 또는 탭)**
- 투자/IR
- 채용
- 파트너십/사업 협력
- 기타

**공통 입력 필드**

| 필드명 | 입력 유형 | 필수 여부 |
|--------|----------|----------|
| 이름 | 텍스트 | 필수 |
| 소속 | 텍스트 | 선택 |
| 이메일 | 이메일 | 필수 |
| 전화번호 | 전화번호 | 선택 |
| 문의 내용 | 텍스트 영역 | 필수 |
| 개인정보 수집/이용 동의 | 체크박스 | 필수 |

**제출 버튼**
문의 보내기

**제출 완료 메시지**
문의가 접수되었습니다. 담당자 확인 후 입력하신 이메일로 회신드리겠습니다.

**유형별 담당자 라우팅 (내부 설정)**
- 투자/IR: BI Group 담당자
- 채용: 경영지원실 인사 담당자
- 파트너십/사업 협력: BI Group 담당자
- 기타: 경영지원실

## 데이터 연동 포인트
- `contact.title`: 페이지 타이틀
- `contact.intro`: 인트로 카피
- `contact.form.types[]`: 문의 유형 목록
- `contact.form.fields[]`: 입력 필드 목록
- `contact.form.submit_label`: 제출 버튼 텍스트
- `contact.form.success_message`: 제출 완료 메시지
- `contact.direct.phone`: 대표 전화
- `contact.direct.email`: 대표 이메일
- `contact.direct.address`: 주소
- `contact.privacy_consent`: 개인정보 수집/이용 동의 문구 (개인정보처리방침 링크 포함)
