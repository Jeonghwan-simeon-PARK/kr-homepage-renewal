# 채용

## 페이지 목적
한국에서 미국 Medicare SaaS를 개발하는 하이케어넷의 일과 팀을 구체적으로 보여주어, 관심 있는 인재가 지원을 결심하도록 유도한다.

## 콘텐츠

### 섹션 (A): 왜 하이케어넷인가

**페이지 타이틀**
하이케어넷에서 일한다는 것

**인트로 카피**
미국 Medicare 시장에서 실제 매출이 발생하는 SaaS 플랫폼을 만드는 팀입니다. 한국에서 미국 의료 시장을 대상으로 제품을 개발하고 운영하는 경험은 흔치 않습니다.

**매력 포인트 4가지 (카드 또는 블록 UI)**

1. 미국 의료 SaaS 개발 경험
미국 공공보험 Medicare 제도 하에서 클리닉이 사용하는 환자 관리 플랫폼(Care Portal)을 직접 만듭니다. CPT 수가 코드 체계, HIPAA 규정, 다국어 고령층 UX 등 일반적인 국내 프로젝트에서는 접하기 어려운 도메인을 다룹니다.

2. 22명 조직, 1인당 임팩트
R&D 10명이 미국 40개 클리닉, 3,000명 이상의 환자가 사용하는 플랫폼을 만들고 운영합니다. 작은 조직이지만 실제 사용자에게 영향을 미치는 범위는 넓습니다.

3. 글로벌 기준의 실무
미국 현지 법인과 협업하며, HIPAA(미국 의료정보 보호법), ISO 27001(정보보안), ISO 27701(개인정보보호) 등 글로벌 보안 기준을 실무에 적용합니다.

4. 해외 매출 비중 96%
내수가 아닌 해외 시장에서 매출이 발생합니다. 만드는 제품이 글로벌 시장에서 직접 검증받는 환경입니다.


### 섹션 (B): 팀 소개

**섹션 타이틀**
함께 일하는 팀

**R&D Group (10명)**
Care Portal과 환자용 앱(Hicare App)을 설계, 개발, 운영하는 팀입니다. 백엔드, 프론트엔드, 모바일, 디자인, 클라우드 인프라, QC까지 소프트웨어 제품의 전 과정을 담당합니다. 전체 직원의 45%가 R&D 인력입니다.

**BI Group (7명)**
미국 시장 영업, 사업 기획, 정부 과제 개발, 임상/제품화 기획을 수행합니다. 미국 클리닉과의 계약 확대, 신규 서비스(APCM, AWV 등) 기획, 파트너십 발굴이 주요 업무입니다.

**경영지원실 (4명)**
인사, 자금 관리, 회계, 총무를 담당합니다. 22명 규모의 조직이 안정적으로 운영될 수 있도록 지원합니다.

**해외지원팀**
미국 현지 법인(캘리포니아) 운영을 지원합니다. 해외법인 경영관리, 인사, 회계 업무를 수행합니다.

### 섹션 (F): 우리가 풀고 있는 기술적 문제

**섹션 타이틀**
우리가 풀고 있는 문제

**인트로 카피**
하이케어넷의 R&D 팀은 미국 의료 시장의 실제 문제를 기술로 해결합니다. 현재 팀이 집중하고 있는 과제입니다.

**문제 1: AI 기반 리포트 자동 생성**
미국 Medicare RPM/CCM 수가 청구에는 CPT 코드별로 정해진 형식의 산출물이 필요합니다. 환자의 생체 데이터와 관리 기록을 분석하여, 각 코드 요건에 맞는 리포트를 자동으로 생성하는 AI 파이프라인을 개발하고 있습니다. 의료진의 행정 부담을 줄이고, 청구 누락을 방지하는 것이 목표입니다.

**문제 2: 다국어 고령층 사용성 최적화**
환자용 앱(Hicare App)의 주 사용자는 65세 이상 고령 환자입니다. 영어, 스페인어, 한국어를 지원하며, 음성 안내 기반 측정 유도와 직관적인 인터페이스를 설계합니다. 다양한 문화권 고령 사용자의 디지털 리터러시 편차를 고려한 UX가 과제입니다.

**문제 3: 실시간 생체 데이터 모니터링 인프라**
3,000명 이상의 환자가 매일 전송하는 혈압, 혈당, 산소포화도 등의 생체 데이터를 실시간으로 수집하고 분석합니다. 비정상 수치를 감지하면 의료진에게 즉시 알림을 보내는 시스템의 안정성과 확장성을 높이는 작업을 진행 중입니다.

## 데이터 연동 포인트
- `careers.hero.title`: 페이지 타이틀
- `careers.hero.subtitle`: 인트로 카피
- `careers.why.items[0-3]`: 4가지 매력 포인트 (제목 + 본문)
- `careers.team.rnd`: R&D Group 소개 텍스트
- `careers.team.bi`: BI Group 소개 텍스트
- `careers.team.admin`: 경영지원실 소개 텍스트
- `careers.team.overseas`: 해외지원팀 소개 텍스트
- `careers.problems.items[0-2]`: 기술적 문제 3건 (제목 + 본문)
- `careers.openings`: 채용 포지션 동적 데이터 (JSON/CMS 연동)
- `careers.apply.always_open`: 상시 지원 안내 카피
