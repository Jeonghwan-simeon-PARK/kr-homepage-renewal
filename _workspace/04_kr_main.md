# 메인 페이지

## 페이지 목적
하이케어넷의 정체성을 즉시 전달하고, 채용 지원자/투자자/파트너 각각이 원하는 페이지로 빠르게 유도한다.

## 콘텐츠

### 섹션 (A): 히어로 영역

**헤드라인**
미국 Medicare 원격환자관리 플랫폼

**서브 카피**
미국 클리닉의 RPM/CCM 수가 청구를 자동화하는 SaaS, Hicare Care Portal.
40개 클리닉, 3,000명 이상의 환자가 사용하고 있습니다.

**CTA 1 (주요)**
사업영역 알아보기

**CTA 2 (보조)**
채용 공고 보기

**비주얼 노트**
배경에 Care Portal 대시보드 목업 이미지를 배치합니다. "소프트웨어 플랫폼 회사"라는 인상이 첫 화면에서 전달되어야 합니다. 의료기기 이미지는 사용하지 않습니다.


### 섹션 (B): 핵심 수치 블록

히어로 바로 아래에 배치합니다. 아이콘 + 숫자 + 설명 레이블 형태입니다.

| 수치 | 레이블 |
|------|--------|
| 40개 | 미국 계약 클리닉 |
| 3,000명+ | 등록 환자 |
| 월 300건+ | 보험 청구 처리 |
| 96% | 해외 매출 비중 |


### 섹션 (C): 타겟별 진입점

3개 카드 형태로 배치합니다. 각 카드에 아이콘, 제목, 설명 1줄, 링크 버튼을 포함합니다.

**카드 1: 사업영역 알아보기**
미국 Medicare 기반 RPM/CCM 플랫폼 사업과 서비스 확장 로드맵을 확인할 수 있습니다.
[사업영역 바로가기]

**카드 2: 합류하기**
22명이 만드는 글로벌 헬스케어 SaaS. 현재 채용 중인 포지션과 팀 소개를 확인하세요.
[채용 페이지 바로가기]

**카드 3: IR 자료 보기**
매출 추이, 시장 기회, 사업 실적 데이터를 제공합니다.
[IR 페이지 바로가기]


### 섹션 (D): 최근 뉴스

**섹션 타이틀**
최근 소식

뉴스룸에서 최신 항목 2~3건을 자동으로 가져옵니다.

표시 형태: 날짜 + 제목 + 요약 1줄

[예시 초기 콘텐츠]
- 2025년: 미국 40개 클리닉 계약 달성
- 2025년: AMC Health MoU 체결
- 2025년: NIPA 미주 ICT 비즈니스 파트너십 참가

[전체 뉴스 보기] 링크


## 필요 이미지
| ID | 설명 | 유형 | 조달 방법 |
|----|------|------|----------|
| IMG-01 | Care Portal 대시보드 목업 (히어로 배경) | Type C: 목업 | 실제 스크린샷 확보 우선, 미확보 시 목업 제작 |
| IMG-02 | 디지털 헬스케어 컨셉 일러스트 (히어로 보조) | Type C: 스톡/AI | 스톡 이미지 또는 AI 생성 |
| IMG-03 | 아이콘 세트 (클리닉, 환자, 보험청구, 매출) | Type B: 디자인 제작 | 브랜드 컬러 적용 아이콘 디자인 |

## 데이터 연동 포인트
- `home.hero.headline`: 헤드라인
- `home.hero.subtext`: 서브 카피
- `home.hero.cta_primary`: CTA 1
- `home.hero.cta_secondary`: CTA 2
- `home.stats.items[0-3]`: 핵심 수치 4개 (숫자 + 레이블)
- `home.entry.items[0-2]`: 타겟별 진입점 3개 (제목 + 설명 + 링크)
- `home.news.items`: 뉴스 동적 데이터 (JSON/CMS 연동, 최신 2~3건)
